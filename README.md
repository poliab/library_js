# Online Library App

Невеликий вебзастосунок для пошуку та перегляду книжок за допомогою відкритого OpenLibrary API. Дозволяє шукати видання за назвою, автором або ключовими словами та переглядати основну інформацію про книгу.

## Можливості

- Пошук книжок за назвою, автором або ключовими словами.
- Перегляд базової інформації про книгу: обкладинка, автор(и), рік публікації.
- Адаптивний інтерфейс, який коректно відображається на комп’ютері, планшеті та телефоні.
- Отримання актуальних даних напряму з публічного OpenLibrary API.

## Технології

- Frontend: чистий JavaScript (ES6+), HTML5, CSS3
- API: [OpenLibrary API](https://openlibrary.org/developers/api) для отримання даних про книжки та авторів
- Архітектура: поділ коду на прості модулі/компоненти для кращої підтримуваності

## Як запустити

Нижче наведено базові кроки, щоб запустити застосунок локально для розробки та тестування.

### Необхідні умови

- Сучасний браузер (Chrome, Firefox, Safari, Edge)
- Будь‑який простий статичний сервер  
  (наприклад, розширення **Live Server** у VS Code або `npx serve` / інший локальний сервер)
