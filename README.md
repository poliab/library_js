# Online Library App

Невеликий вебзастосунок для пошуку та перегляду книжок за допомогою відкритого OpenLibrary API. Дозволяє шукати видання за назвою, автором або ключовими словами та переглядати основну інформацію про книгу.

## Можливості

- Пошук книжок за назвою, автором або ключовими словами.
- Перегляд базової інформації про книгу: обкладинка, автор(и), рік публікації.
- Адаптивний інтерфейс, який коректно відображається на комп’ютері, планшеті та телефоні.
- Отримання актуальних даних напряму з публічного OpenLibrary API.

## Технології

- Frontend: чистий JavaScript (ES6+), HTML5, CSS3
- API: [OpenLibrary API](https://openlibrary.org/developers/api) для отримання даних про книжки та авторів
- Архітектура: поділ коду на прості модулі/компоненти для кращої підтримуваності

## Як запустити

Нижче наведено базові кроки, щоб запустити застосунок локально для розробки та тестування.

### Необхідні умови

- Сучасний браузер (Chrome, Firefox, Safari, Edge)
- Будь‑який простий статичний сервер  
  (наприклад, розширення **Live Server** у VS Code або `npx serve` / інший локальний сервер)

### Встановлення

1. Клонувати репозиторій:
   ```bash
   git clone https://github.com/yourusername/online-library-app.git
   ```

2. Перейти в директорію проєкту:
   ```bash
   cd online-library-app
   ```

3. Запустити локальний сервер.  
   Наприклад, якщо встановлено Node.js:
   ```bash
   npx serve .
   ```
   Після запуску застосунок буде доступний за адресою `http://localhost:3000` або за портом, який вкаже ваш сервер.

## Як користуватися

Після запуску застосунку локально:

1. Введіть назву книги або автора в поле пошуку та натисніть Enter / кнопку пошуку.
2. Перегляньте список знайдених книжок у вигляді карток.
3. Натисніть на конкретну картку, щоб побачити детальнішу інформацію про книгу.
4. Скористайтеся елементами навігації або кнопкою «Назад», щоб повернутися до результатів пошуку.

## Структура проєкту

```text
├── src/
│   ├── api/          # Запити до OpenLibrary API
│   ├── components/   # Прості UI‑компоненти (наприклад, картка книги, пошуковий інпут)
│   ├── styles/       # CSS‑стилі
│   └── main.js       # Точка входу, ініціалізація застосунку
├── index.html        # Основний HTML‑шаблон
└── README.md         # Цей файл з описом проєкту
```

(За потреби зміни назви папок/файлів під фактичну структуру.)

## Налаштування

Застосунок використовує публічні, неавторизовані endpoints OpenLibrary.  
Додаткові змінні середовища чи API‑ключі не потрібні. Для коректної роботи потрібне активне підключення до інтернету.

## Плани розвитку

У найближчих планах:

- Додати пагінацію або нескінченну прокрутку для великих результатів пошуку.
- Реалізувати список «Обране» / «Хочу прочитати» з використанням `localStorage`.
- Додати фільтри за роком, мовою чи типом видання.
- Покращити структуру стану, розділити логіку та UI‑компоненти.
- Додати базові тести для ключових частин логіки (пошук, робота з API).
